Manual de Uso HIT

¿Qué es HIT?

HIT es una plataforma educativa que permite gestionar contenidos académicos, subir tareas, visualizar materiales y organizar evidencias de aprendizaje dentro del entorno universitario.

Pasos para utilizar HIT

1. Ingresar al portal institucional HIT. 2. Introducir usuario y contraseña proporcionados por la universidad. 3. Seleccionar el curso correspondiente. 4. Descargar materiales y revisar contenidos. 5. Subir tareas y evidencias solicitadas.

Ir a Plataforma HIT

Manual de Uso HIT HUB (GitHub)

¿Qué es GitHub?

GitHub es una plataforma que permite almacenar proyectos, controlar versiones del código y compartir desarrollos tecnológicos con otros usuarios.

Pasos para usar GitHub

1. Crear una cuenta en GitHub. 2. Crear un repositorio nuevo. 3. Subir archivos al repositorio. 4. Actualizar proyectos mediante commits. 5. Compartir el enlace del repositorio.
