Manual de Uso Git y GitHub

¿Qué es Git?

Git es un sistema de control de versiones que permite registrar los cambios realizados en el código fuente.

¿Qué es GitHub?

GitHub es una plataforma en línea que permite almacenar repositorios y compartir proyectos.
